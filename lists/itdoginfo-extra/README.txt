# Дополнительные домены и подсети

Файлы в этой папке — ваши дополнения к спискам из itdoginfo.
Содержимое НЕ перезаписывается при синхронизации.

Формат: один домен или подсеть (CIDR) на строку.
Комментарии: строки, начинающиеся с #

Имена файлов соответствуют itdoginfo:
  Anime.txt, Block.txt, GeoBlock.txt, News.txt, Porn.txt, HODCA.txt
  Cloudflare.txt, Discord.txt, Discord-Subnets.txt, HDRezka.txt
  Meta.txt, Meta-Subnets.txt, Telegram.txt, Telegram-Subnets.txt
  TikTok.txt, Twitter.txt, Twitter-Subnets.txt, YouTube.txt
