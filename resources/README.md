# Ресурсы ALLOW

- **allow** — init.d, manage.d (скрипты управления, autostart, dns-mode и т.д.).
- **dnsmasq-full** — конфиги, init.d, ipsets, sync-allow-lists.
- **markalltovpn** — route-by-mark.sh, NDM netfilter (hotspot VPN по марке).
- **monitor** — веб-монитор (API, статика, шаблоны). Sing-box: выборочный роутинг пакетов по марке, кнопка включить/выключить.
- **sing-box** — конфиг, init.d, netfilter.
- **stubby** — конфиги DoT, init.d.

Файл в кодировке UTF-8.
